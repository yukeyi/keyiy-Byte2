Accountability: About This Data Set

This dataset consists of the performance of 526 football players in season 15-16. Besides personal information: players' name, value and their team, main data is matrics of their performance, including appearance and substitution time, number of goals, number of penalties they miss, number of red and yellow card they get, etc. THere are also some data that only owned bt defender of GK player like number of goal conceded(GC). Based on performance data, dataset also provides a total points computed based on single performance item and average points computed by Total / Value.

Where did this data come from?

This data is about 526 football player's performance informantion on Telegraph Fantasy Football Premier League 2015-16. It comes from google public data tables.

Original website : https://fantasy.premierleague.com/

Who has access to it?

The data is provided by Fantasy Football Company and is free to use as on-line dataset. Also I set the dataset as public, so everyone who has the url can see it but cannot modify it.
